📘 About this project

✅ Live app with a one-click demo dataset

The dashboard reads contract award data - such as the country datasets published by ProAct — and produces an interactive analysis across three modules:

  • Economic Outcomes — market sizing, supplier dynamics and entry, relative prices against estimates, competition and single bidding
  • Administrative Efficiency — procedure mix, value bunching below thresholds, submission and decision periods, and regression evidence linking administrative shortcuts to reduced competition
  • Procurement Integrity — missing-data forensics, buyer–supplier concentration over time, unusual market-entry networks, and robustness-checked integrity regressions

Every figure is exportable individually, as a figures ZIP, or inside an auto-generated Word report per section. A fully synthetic demo dataset (fictional country Demoland, 15,674 awards, 2015–2024) ships with the app so anyone can explore every feature in one click.

🚀 Try it

Open the live app and click Run Demo on the Setup page — the bundled dataset loads with every threshold pre-configured. Or upload ProAct country dataset. You can also upload your own data - just make sure its structure and variable names follow data dictionary requirements

📈 Run it locally

To run the dashboard on your own machine you will need:

  • R version 4.3 or newer, with RStudio (Posit) recommended
  • The repository — clone or download it from GitHub
  • Open R in the repository folder and run shiny::runApp() — required packages are listed in the README
  • Click Run Demo on the Setup tab, or upload a ProAct dataset
View Repository on GitHub

Repository structure

  • global.R, ui.R, server.R — the Shiny app (multi-file layout)
  • utils_shared.R, econ_out_utils.R, admin_utils.R, integrity_utils.R — analysis modules
  • demo-data/ — bundled synthetic dataset, guide, and generator
  • docs/ — this documentation site
  • deploy_shinyapps.R, PUBLISHING.md — deployment kit
